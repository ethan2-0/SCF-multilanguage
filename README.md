==========================================
========Serengetti  Config  Format========
==========================================
Serengetti Config Format (a.k.a. SCF) is a simple name-
value pair config format, like the following:
----------customer-ethan20.scf------------
username:ethan20
password:6B69E32109F768E73D57A366F64B4324
email:ethan@serengetti-software.com
------------------------------------------
+-----------------WARNING----------------+
|                                        |
|  DO NOT leave a space after the name   |
|  in the name/value pair.               |
|                                        |
+----------------------------------------+
It is available in many differrent languages, including:
• .NET (or as a DLL build from .NET for any system that 
will accept it)
