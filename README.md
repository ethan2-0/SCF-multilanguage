===Serengetti  Config  Format===<br/>
Serengetti Config Format (a.k.a. SCF) is a simple name-
value pair config format, like the following:<br/>
username:ethan20<br/>
password:6B69E32109F768E73D57A366F64B4324<br/>
Warning: Do not leave a space after the name in a name-
value pair unless you intend it to be in the value.<br/>
It is available in many differrent languages, including:<br/>
• .NET (or as a DLL build from .NET for any system that 
will accept it)
